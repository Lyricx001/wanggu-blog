WANGGU 个人博客使用说明
========================

一、部署到你现有的 Cloudflare Pages 项目
1. 登录 Cloudflare。
2. 进入“Workers 和 Pages”。
3. 点击你的项目“wangzhan”。
4. 点击右上角“创建部署”。
5. 上传 wanggu-personal-blog.zip。
6. 等待出现绿色对勾。
7. 打开 https://wanggu.top 刷新查看。

二、修改网站名称
打开 index.html，搜索：
WANGGU BLOG

把它改成你想要的网站名称即可。

三、修改首页介绍
打开 index.html，搜索：
记录技术、跨境电商

修改对应的标题和介绍文字。

四、修改“关于我”
打开 index.html，搜索：
你好，我是这个博客的作者

修改下面的个人介绍内容。

五、添加或修改文章
打开 posts.js。

每篇文章的结构如下：

{
  id: "文章英文标识",
  title: "文章标题",
  excerpt: "文章摘要",
  category: "文章分类",
  date: "2026-07-03",
  readTime: "5 分钟",
  featured: true,
  content: `
    <p>文章正文</p>
    <h2>小标题</h2>
    <p>更多正文</p>
  `
}

featured: true 代表显示在首页精选文章中。
featured: false 代表只显示在“全部文章”页面。

六、重新制作压缩包
1. 打开网站文件夹。
2. 按 Ctrl+A 选中里面所有文件。
3. 鼠标右键，选择“压缩为 ZIP 文件”。
4. 打开 ZIP 后，第一层必须直接看到 index.html。

注意：
不要把外层文件夹一起压缩，否则 Cloudflare 可能无法找到首页。


七、动态效果增强版新增内容
- 首次打开加载动画
- 页面切换淡入动画
- 滚动进度条
- 鼠标跟随光晕
- 轻量动态粒子背景
- 首页插画鼠标视差
- 卡片 3D 倾斜与高光
- 内容滚动出现动画
- 按钮磁吸和点击水波纹
- 导航栏滚动悬浮效果
- 自动兼容手机和“减少动态效果”设置

此版本的文件名：
wanggu-personal-blog-dynamic.zip

部署方式和原版相同，直接在 Cloudflare Pages 中创建新部署并上传整个 ZIP 文件。


八、给文章添加封面图片
1. 打开 Pages CMS。
2. 进入“文章管理”。
3. 点击某篇文章，或者新建文章。
4. 找到“封面图片”。
5. 上传图片并保存。
6. 首页精选卡片、文章列表和文章详情页都会显示封面。

九、给文章正文添加图片
1. 打开文章的“文章正文”编辑器。
2. 在需要插入图片的位置点击图片按钮，或直接拖入图片。
3. 图片会自动上传到 media/posts 目录。
4. 点击保存。
5. 等待 Cloudflare 自动部署完成。
6. 刷新网站即可看到图片。

十、正文图片效果
- 正文图片会自动适配宽度
- 自动带圆角和阴影
- 点击正文图片可以放大预览
- 按 Esc 或点击黑色背景可关闭预览

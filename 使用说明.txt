WANGGU BLOG 后台管理版
=======================

GitHub 用户名：Lyricx001
建议仓库名称：wanggu-blog
仓库地址：https://github.com/Lyricx001/wanggu-blog

这个版本已经接入 Pages CMS：
- 可以在网页后台新建、修改和删除文章
- 可以修改网站名称、首页介绍和“关于我”
- 可以上传文章封面和正文图片
- 保存后自动写入 GitHub
- Cloudflare Pages 会自动重新部署网站

重要：
1. 先解压这个 ZIP。
2. 上传“解压后里面的全部文件”到 GitHub 仓库。
3. 不要只把 ZIP 文件上传到 GitHub。
4. .pages.yml 是隐藏配置文件，不能删除。

Pages CMS 后台：
https://app.pagescms.org

Cloudflare Pages Git 部署建议：
- 生产分支：main
- 框架预设：无
- 构建命令：exit 0
- 构建输出目录：/
- 根目录：留空

文章后台字段：
- 文章链接标识：只能写小写英文、数字、短横线
- 文章标题
- 摘要
- 分类
- 日期
- 阅读时间
- 首页精选
- 封面图片
- 正文

保存文章后，等待 Cloudflare 部署完成，网站会自动更新。


八、给文章添加封面图片
1. 打开 Pages CMS。
2. 进入“文章管理”。
3. 点击某篇文章，或者新建文章。
4. 找到“封面图片”。
5. 上传图片并保存。
6. 首页精选卡片、文章列表和文章详情页都会显示封面。

九、给文章正文添加图片
1. 打开文章的“文章正文”编辑器。
2. 在需要插入图片的位置点击图片按钮，或直接拖入图片。
3. 图片会自动上传到 media/posts 目录。
4. 点击保存。
5. 等待 Cloudflare 自动部署完成。
6. 刷新网站即可看到图片。

十、正文图片效果
- 正文图片会自动适配宽度
- 自动带圆角和阴影
- 点击正文图片可以放大预览
- 按 Esc 或点击黑色背景可关闭预览



功能升级说明
============

本升级包新增：
1. 全站搜索：顶部搜索按钮，支持搜索标题、分类、摘要、正文。快捷键 Ctrl+K 或 /
2. 文章目录：文章中有两个以上 h2/h3 小标题时自动生成目录。
3. 文章阅读工具：复制文章链接、字体放大、字体缩小。
4. 返回顶部：向下滚动后右下角出现返回顶部按钮。
5. 阅读进度条：页面顶部显示阅读进度。
6. 公告弹窗：可以在 Pages CMS 的“网站设置”中开启并修改标题、内容和按钮。
7. 后台功能开关：搜索、阅读进度、返回顶部、文章目录、公告弹窗都能在后台开关。

覆盖方式：
1. 解压此 ZIP。
2. 上传里面的 .pages.yml、index.html、style.css、script.js 到 GitHub 仓库根目录。
3. Commit changes。
4. 等待 Cloudflare 自动部署。
5. 刷新 Pages CMS 后可在“网站设置”看到新增功能开关。

不会覆盖 posts.json、site.json、media。



继续功能升级说明
================

本升级包新增：
1. 文章归档页
   - 顶部新增“归档”
   - 自动按年份、月份列出所有文章
   - 显示文章总数、分类数量、归档年份
2. 本地收藏
   - 顶部新增“收藏”
   - 文章页可点击“收藏文章”
   - 收藏只保存在当前浏览器，不需要登录
3. 相关推荐
   - 文章详情页底部自动推荐同分类文章
4. 上一篇 / 下一篇
   - 文章详情页底部自动显示继续阅读
5. 文章分享
   - 支持系统分享；不支持时自动复制链接
6. 打印 / 保存 PDF
   - 文章页点击“打印 / PDF”
   - 可在浏览器打印窗口选择“另存为 PDF”
7. 代码复制
   - 正文里有代码块时，自动出现“复制代码”按钮
8. 后台功能开关
   - 归档页
   - 本地收藏
   - 相关推荐
   - 上一篇下一篇
   - 代码复制
   - 打印/PDF按钮

覆盖方式：
1. 解压本 ZIP。
2. 上传里面的 .pages.yml、index.html、style.css、script.js 到 GitHub 仓库根目录。
3. Commit changes。
4. 等待 Cloudflare 自动部署。
5. 刷新 Pages CMS，进入“网站设置”即可看到新增开关。

不会覆盖：
- posts.json
- site.json
- media 图片文件

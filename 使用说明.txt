WANGGU BLOG 后台管理版
=======================

GitHub 用户名：Lyricx001
建议仓库名称：wanggu-blog
仓库地址：https://github.com/Lyricx001/wanggu-blog

这个版本已经接入 Pages CMS：
- 可以在网页后台新建、修改和删除文章
- 可以修改网站名称、首页介绍和“关于我”
- 可以上传文章封面和正文图片
- 保存后自动写入 GitHub
- Cloudflare Pages 会自动重新部署网站

重要：
1. 先解压这个 ZIP。
2. 上传“解压后里面的全部文件”到 GitHub 仓库。
3. 不要只把 ZIP 文件上传到 GitHub。
4. .pages.yml 是隐藏配置文件，不能删除。

Pages CMS 后台：
https://app.pagescms.org

Cloudflare Pages Git 部署建议：
- 生产分支：main
- 框架预设：无
- 构建命令：exit 0
- 构建输出目录：/
- 根目录：留空

文章后台字段：
- 文章链接标识：只能写小写英文、数字、短横线
- 文章标题
- 摘要
- 分类
- 日期
- 阅读时间
- 首页精选
- 封面图片
- 正文

保存文章后，等待 Cloudflare 部署完成，网站会自动更新。


八、给文章添加封面图片
1. 打开 Pages CMS。
2. 进入“文章管理”。
3. 点击某篇文章，或者新建文章。
4. 找到“封面图片”。
5. 上传图片并保存。
6. 首页精选卡片、文章列表和文章详情页都会显示封面。

九、给文章正文添加图片
1. 打开文章的“文章正文”编辑器。
2. 在需要插入图片的位置点击图片按钮，或直接拖入图片。
3. 图片会自动上传到 media/posts 目录。
4. 点击保存。
5. 等待 Cloudflare 自动部署完成。
6. 刷新网站即可看到图片。

十、正文图片效果
- 正文图片会自动适配宽度
- 自动带圆角和阴影
- 点击正文图片可以放大预览
- 按 Esc 或点击黑色背景可关闭预览
